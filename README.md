# 📦 Nexventory

> Sistem Informasi Manajemen Inventaris berbasis web yang saya bangun secara mandiri dari nol dalam waktu kurang lebih satu minggu menggunakan Laravel.

## 🌐 Demo

- Website : https://nexventory-production.up.railway.app
- GitHub : https://github.com/Argaa9/nexventory

---

## 🚀 Fitur

- 🔐 Login, Register, Logout
- 👥 Role Admin, Manager, Staff
- 📦 CRUD Data Barang
- 🖼️ Upload Gambar Barang
- 📂 Manajemen Kategori
- 📋 Peminjaman Barang
- 📝 Riwayat Peminjaman
- 📉 Monitoring Stok Menipis
- 📊 Dashboard Statistik
- 📄 Export PDF
- 📊 Export Excel
- ⭐ Testimonial

---

## 🛠️ Teknologi

- Laravel 12
- PHP 8.2
- MySQL
- Tailwind CSS
- Railway
- GitHub

---

## ⚙️ Instalasi

```bash
git clone https://github.com/Argaa9/nexventory.git

cd nexventory

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan storage:link

php artisan serve
```

Import database:

```
nexventory.sql
```
